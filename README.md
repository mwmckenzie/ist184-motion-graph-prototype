# IST-184 Motion Graph Vis Prototype

https://mwmckenzie.github.io/ist184-motion-graph-prototype/

## Legend
### Dots
- Blue: AIS Traffic (Proportional To Observed Numbers)
- Orange: Waypoint Locations (Observed)
- Purple: Port Locations

## Basic Controls
### Panning
- Up: W
- Down: S
- Left: A
- Right D

### Zoom
- Mousewheel
